# SixPack30 

30 günlük profesyonel karın kası antrenman programı. Şık tasarımı ve detaylı takip özellikleriyle hedefinize ulaşmanız için tasarlandı.

---

### Özellikler
- **Antrenman Takibi**: Günlük egzersiz süreçleri ve yakılan kalori analizi.
- **Dönüşüm Verileri**: Kilo değişimi, yağ oranı ve kas kütlesi artışı için grafiksel gösterim.
- **Modern Arayüz**: Montserrat font ve özel ikon setleriyle premium kullanıcı deneyimi.
- **Gelişmiş Profil**: Yaş, boy ve kilo bilgilerine göre kişiselleştirilmiş metrikler.

---

### Teknik Detaylar
- **Framework**: Flutter
- **State Management**: Riverpod
- **Tasarım**: Pixel-perfect responsive layout (flutter_screenutil)
- **Assetler**: Özel iconstack.io varlıkları ve özgün PNG ikonlar.

---

### Kurulum
```bash
git clone https://github.com/flyworkteam/sixpack30.git
cd sixpack30
flutter pub get
flutter run
```
